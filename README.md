# SignVerse App

Flutter mobile app for real-time sign language translation.

## Stack
- **Flutter** — cross-platform mobile framework
- **Riverpod** — state management
- **GoRouter** — navigation
- **Dio** — HTTP client for API calls

## Setup

1. Clone the repo
2. Run `flutter pub get`
3. Create `lib/core/constants/api_constants.dart` with your Railway API URL
4. Run on device: `flutter run`
